## Slide 2: Problem Statement
- Farmers lack access to timely agricultural advice and resources
- Difficulty connecting with experts and managing crop data
- AgroConnect bridges this gap with a digital solution

---

## Slide 3: Solution Overview
- A web platform for farmers and agricultural experts
- Features:
  - User registration and login
  - Crop data management
  - Expert query submission (optional)
  - Dashboard for insights

---

## Slide 4: Tech Stack
| Layer       | Technology         |

| Backend     | Node.js + Express  |
| Database    | MongoDB            |
| Auth        | JWT                |
| CI/CD       | GitHub Actions     |
